# BlogApp
Vue CLI node js app that allows users to perform all CRUD functionalities on the app and search for their blog posts
  # features
  1.CRUD Functionalities:
  -Create a post:
  Fill in a title and content in the form at the top of the page.
  Click Publish Post to save the new post to the database.
  The post appears instantly at the top of the list without needing a page refresh.
  -Read a post
  -Edit a post :
    Click the Edit button on any post to switch it into edit mode.
    The title and content become editable fields inline.
    Click Save Changes to persist the updated post to MongoDB.
  -Delete a post :
    Click the Delete button to permanently remove a post from the database.
    The post is immediately removed from the displayed list.

   # Tech stack
      Frontend - VUE 3:Composition AI
      Backend - Node js, express
      Database - MongoDB(Mongoose)
      HTTP - Fetch API
